# Chapter 1: Introduction — Statistics (Witte & Witte, 11th Ed.)

## Overview

Chapter 1 introduces the field of statistics and the foundational concepts you need for the rest of the course. The central theme is **variability**—statistics exists because observations in the real world differ from one another. The chapter explains why we study statistics, what statistics is, and the key terminology used throughout the book.

---

## Chapter Summary & Explanation

### 1.1 Why Study Statistics?

Statistics helps you:

- **Understand everyday claims** — Phrases like "on average," "no evidence of a relationship," and "significantly more" appear in news and research. With statistics, you can interpret and question these claims.
- **Read research reports** — Results sections use symbols such as `p < .05`, `t(48) = 3.21`, and `χ²(4, n = 185) = 11.49`. Statistics lets you decode and evaluate these findings.
- **Plan your own research** — You can design and analyze modest studies and know when to consult a statistician for more complex work.

---

### 1.2 What Is Statistics?

Statistics has two main branches:

| Branch | Purpose | Tools |
|--------|---------|-------|
| **Descriptive Statistics** | Organize and summarize actual observations | Tables, graphs, averages, ranges, correlations |
| **Inferential Statistics** | Generalize beyond actual observations | Hypothesis tests, confidence intervals, estimates |

**Examples:**
- **Descriptive:** A table of romantic affairs reported by students; a graph of global temperature over 30 years.
- **Inferential:** A pollster’s claim about voter preferences; a researcher’s hypothesis about meditators vs. non-meditators.

---

### 1.3 More About Inferential Statistics

#### Populations and Samples

- **Population** — Any complete collection of observations or potential observations (e.g., all U.S. college students’ anxiety scores).
- **Sample** — A smaller collection of actual observations drawn from a population (e.g., 100 students’ anxiety scores).

The same data can be a population or a sample depending on the question. For example, weights of 53 male statistics students can be:
- A **population** if you care about that specific group (e.g., boat capacity).
- A **sample** if you want to generalize to all male college students.

#### Random Sampling (Surveys)

- **Random sampling** — Each potential observation in the population has an equal chance of being selected.
- Used in **surveys** to generalize from a sample to a population.
- Example: Lottery numbers; random-digit dialing for opinion polls.

#### Random Assignment (Experiments)

- **Random assignment** — Each person has an equal chance of being assigned to any group.
- Used in **experiments** to decide whether differences between groups are real or due to chance.
- Example: Assigning volunteers to treatment vs. control groups.

#### Surveys vs. Experiments

| Surveys | Experiments |
|---------|-------------|
| Use random **sampling** | Use random **assignment** |
| Generalize from sample to population | Decide if group differences are real or transitory |
| Focus on representation | Focus on cause-effect |

### 1.4 Three Types of Data

| Type | Description | Example |
|------|-------------|---------|
| **Qualitative** | Words, letters, or codes representing classes/categories | Yes/No, Y/N, ethnic group |
| **Ranked** | Numbers representing relative standing | 1st, 2nd, 3rd place |
| **Quantitative** | Numbers representing an amount or count | Weight (lbs), IQ score, family size |

**Tip:** Look at a single observation. Does it represent a category, a rank, or an amount?

---

### 1.5 Levels of Measurement

Each type of data corresponds to a level of measurement:

| Data Type | Level | Key Property | Arithmetic |
|-----------|-------|--------------|------------|
| **Qualitative** | **Nominal** | Classification only | Meaningless |
| **Ranked** | **Ordinal** | Order (relative standing) | Limited |
| **Quantitative** | **Interval/Ratio** | Equal intervals + true zero | Full range |

#### Nominal Measurement
- Only classification (e.g., male/female, mood disorders).
- Codes like 1 and 2 do not imply “twice as much.”
- Averages are not meaningful.

#### Ordinal Measurement
- Reflects order (e.g., 1st, 2nd, 3rd place).
- Distances between ranks are not necessarily equal.
- Mean of ranks 1 and 3 is not necessarily rank 2.

#### Interval/Ratio Measurement
- **Equal intervals** — Same difference = same amount (e.g., 120–130 lbs = 130–140 lbs).
- **True zero** — Zero means complete absence (e.g., weight).
- Supports meaningful arithmetic (means, ratios).

**Note:** Temperature in Fahrenheit has equal intervals but no true zero (0°F ≠ no heat). IQ and similar scales are often treated as **approximately interval**.

---

### 1.6 Types of Variables

#### Variable vs. Constant
- **Variable** — A characteristic that can take different values (e.g., weight, Yes/No).
- **Constant** — A characteristic that takes only one value (a single observation).

#### Discrete vs. Continuous

| Type | Description | Examples |
|------|-------------|----------|
| **Discrete** | Isolated numbers with gaps | Number of children, countries visited |
| **Continuous** | Numbers with no theoretical gaps | Weight, reaction time, SAT score |

**Approximate numbers:** Continuous variables are rounded (e.g., 150 lbs = 149.5–150.5 lbs). These are approximate, not exact.

#### Independent vs. Dependent Variables

| Variable | Role | Example |
|----------|------|---------|
| **Independent** | Manipulated by the investigator | Type of training (active listening vs. none) |
| **Dependent** | Measured/recorded as outcome | Number of communication breakdowns |

#### Experiments vs. Observational Studies

| Study Type | Manipulation | Cause-Effect |
|------------|--------------|--------------|
| **Experiment** | Investigator assigns treatment | Stronger conclusions |
| **Observational Study** | No manipulation; variables observed | Weaker; correlation only |

#### Confounding Variable
- An uncontrolled variable that co-varies with the independent variable.
- Can make cause-effect conclusions unreliable.
- **Reduced by:** Random assignment and standardizing conditions.

## Important Terms (Glossary)

| Term | Definition |
|------|------------|
| Descriptive statistics | Organizing and summarizing actual observations |
| Inferential statistics | Generalizing beyond actual observations |
| Population | Complete collection of observations or potential observations |
| Sample | Smaller collection of actual observations from a population |
| Random sampling | Each population member has equal chance of being selected |
| Random assignment | Each person has equal chance of being in any group |
| Data | Collection of actual observations or scores |
| Qualitative data | Words, letters, or codes representing categories |
| Ranked data | Numbers representing relative standing |
| Quantitative data | Numbers representing amount or count |
| Level of measurement | Extent to which numbers represent attributes |
| Nominal measurement | Classification only |
| Ordinal measurement | Order/relative standing |
| Interval/ratio measurement | Equal intervals and true zero |
| Variable | Characteristic that can take different values |
| Constant | Characteristic that takes only one value |
| Discrete variable | Isolated numbers with gaps |
| Continuous variable | Numbers with no theoretical gaps |
| Approximate numbers | Rounded values (e.g., continuous variables) |
| Independent variable | Treatment manipulated by investigator |
| Dependent variable | Outcome measured by investigator |
| Experiment | Study where investigator assigns treatment |
| Observational study | Study with no manipulation of variables |
| Confounding variable | Uncontrolled variable that compromises interpretation |
